Guest lectures and guided computer exercises will enable participants to apply a variety of methods to the generation and analysis of GBS data. We will also cover variant detection and quality control for SNP calling. Guest lecturers and guided computer exercises will expose participants to practical data management, data quality assessment, visualization tools, and the statistics underlying genotyping-by-sequencing. This class will not cover population genetics in any detail, but will focus more on the preparation of data for downstream analysis from laboratory and field studies.</p> <p><b>Course Specific Measureable Student Learning Outcomes:</b></p> <p>At the end of this course, you should be able to:</p> <ol style="list-style-type: decimal"> <li>Interact more efficiently with core lab personnel on designing and implementing GBS experiments.</li> <li>Utilize the cloud computing environment at the Center for Genome Research and Biocomputing at Oregon State University.</li> <li>Run scripts interactively and in batch using command line for GBS Analysis.</li> <li>Utilize R-scripts for experimental design before GBS and data visualization afterwards.</li> </ol> <p><b>Learning Resources:</b><br /> There is no textbook.
